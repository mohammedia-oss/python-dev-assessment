# Python Developer Assessment

## Overview

This repository contains my solutions for the Python Developer Assessment Program. It demonstrates Python fundamentals, Git/GitHub workflow, object-oriented programming, error handling, data structures, algorithms, and API interaction.

## Tasks Completed

- Task 1.1 – Hello World (`hello.py`)
- Task 1.2 – Git & GitHub Basics
- Task 1.3 – Code Style & Linting (`bad_style.py`)
- Task 2.1 – Data Structures & Algorithms (`dsa_challenges.py`)
- Task 2.2 – Object-Oriented Programming (`book_store.py`)
- Task 2.3 – Error Handling & Debugging (`debug_errors.py`)
- Task 3.1 – API Interaction (`api_client.py`)

## How to Run

Open a terminal in this project folder and run:

```bash
python hello.py
```

Replace `hello.py` with any of the other Python files if you want to run a different task.
